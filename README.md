Flower Store Project | Front-End ApplicationReact

A modern, responsive flower store web application, designed and developed with React and TypeScript.
The project focuses on user experience, state management, and clean component-based architecture.

Main Features:

-Product catalog with dynamic filtering and sorting

-Add, edit, and manage flower items in the store

-User management and form validation using Formik and Yup

-Fully responsive design built with Bootstrap and Material UI

-Redux for global state management and Axios for API communication

-Custom Hooks and Lazy Loading for optimized performance

Key Focus:

-Building a scalable and maintainable React application

-Writing clean, reusable, and typed code with TypeScript

-Implementing best practices in state management and form validation

-Prioritizing UI/UX with smooth, responsive design

Technologies:
React 18, TypeScript, Redux Toolkit, Bootstrap, Material UI (MUI), Formik, Yup, Axios, Custom Hooks, Lazy Loading
